# 🧬 EvoSim — Simulazione di Vita Artificiale

Un ecosistema in un singolo file HTML. Delle creature con un **genoma**
(velocità, raggio visivo, dimensione, colore) si muovono in un ambiente 2D,
cercano cibo, accumulano energia e si **riproducono** generando figli con
piccole **mutazioni** casuali. Chi non trova cibo muore. Nessun comportamento
è programmato: le strategie **emergono** dalla selezione naturale.

## Come usarlo

Apri `index.html` in un qualsiasi browser (funziona anche da telefono).

- **Play / Pausa / Reset** per controllare la simulazione
- Slider per **velocità**, **crescita del cibo** e **tasso di mutazione**
- **Clicca (o tocca) nell'ambiente** per aggiungere cibo dove vuoi
- I grafici mostrano la **popolazione** e i **tratti medi** nel tempo:
  è lì che si legge l'evoluzione

## Pubblicazione (GitHub Pages)

Il sito è pubblicato automaticamente da GitHub Pages a partire da questo
repository. Ogni modifica al file `index.html` aggiorna il sito online.
